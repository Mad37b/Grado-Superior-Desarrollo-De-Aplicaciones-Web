# Grado-Superior-Desarrollo-De-Aplicaciones-Web
En Este Repositorio Orientado al Grado Superior de Desarrollo de Aplicaciones web, Guardaremos los ejercicios y apuntes realizados entre el curso de 2023 hasta 2026 lo mejor explicado posible
para que todo quien encuentre este repositorio pueda ayudarse mejor con el desarrollo de su curso

Estos apuntes y ejercicios estan orientados además con la Formación Dual y la nueva ley de educación LOMLOE (Ley Orgánica 3/2020, de 29 de diciembre, de Educación)

Ley LOMLOE: https://www.educacionfpydeportes.gob.es/biblioteca-central/blog/2022/lomloe.html

El grado superior a sido realizado entre el Instituto I.E.S Val do Asma por la Formación dual y el C.I.F.P A Carballeira

Cuenta con 16 materias a realizar desde el año 2024 en presencial y en 2025 en distancia modular según la experiencia en la Xunta de Galicia, las siguientes materias ordenadas en el ámbito ordinario son las siguientes( el orden de las materias en **Distancia modular** varian según tu elección al curso):

# 1º Curso 

- Programación - Java y JavaFX/SWING y Jakarta o Java web

- Base de datos - Java y oracle SQL y mongodb con java, SQLPL

- Lenguaje de marcas - XML y Json , XSL 

- Sistemas informáticos

- Entornos de desarrollo 

- Inglés

- Sostenibilidad

 - Aplicacion de Servidor - Jakarta 9.0 / 10 - (Java web - JSP )

 - IPE 1 / FOL 

# 2º Curso 

 - Digitalización

 - Habilidades de lenguaje extranjera

 - Entorno Cliente - Javascript 

 - Entorno Servidor - PHP

 - Aplicaciones web - Apache ,FTP , Tomcat , wildfly

 - IPE 2 / EMPRESA y emprendimiento

 - Diseño web  - CSS y HTML5
